# 機能要件

## MVP機能一覧

### 1. 認証
- メール/パスワードによるサインアップ・ログイン
- ログアウト
- セッション管理（自動更新）

### 2. 習慣管理（CRUD）
- 習慣の作成（名前、カテゴリ、目標タイプ、目標値）
- 習慣の編集・削除
- 習慣のアーカイブ（論理削除）
- カテゴリによるフィルタリング

### 3. 日次記録
- 完了チェック（boolean型習慣）
- 数値入力（count型習慣：読書ページ数、運動時間など）
- 日付ごとの記録

### 4. ダッシュボード
- 今日の習慣リスト（未完了/完了状態）
- 週間進捗グラフ（Recharts）
- 現在のストリーク表示

### 5. カレンダービュー
- 月間ヒートマップ（達成度を色で表現）
- 日付クリックで詳細表示

## 画面構成

| パス | 画面名 | 説明 |
|------|--------|------|
| `/` | ランディング | 未認証時のトップページ |
| `/auth/login` | ログイン | メール/パスワード認証 |
| `/auth/signup` | サインアップ | 新規登録 |
| `/dashboard` | ダッシュボード | 今日の習慣、ストリーク、週間グラフ |
| `/habits` | 習慣一覧 | 習慣のCRUD管理 |
| `/habits/new` | 習慣作成 | 新規習慣追加フォーム |
| `/habits/[id]` | 習慣詳細 | 編集、統計、削除 |
| `/calendar` | カレンダー | 月間ヒートマップ |
| `/settings` | 設定 | プロフィール、テーマ |

## 非機能要件

- **レスポンシブ:** モバイルファースト設計
- **言語:** 日本語UI
- **テーマ:** ダークモード対応
- **パフォーマンス:** LCP < 2.5s
